# 🧪 Interactive Lab Modules Collection

A modular, plug-and-play toolkit designed to accelerate the development of interactive 3D environments in Unity. This repository contains self-contained systems that handle user interface feedback and dynamic audio management without requiring complex hardcoding for every object.

These modules are built to be highly scalable, making them perfect for complex interactive projects like virtual laboratories, puzzle games, or educational simulations.

## 📦 Included Modules

Each system in this collection is fully decoupled. You can drop them into any existing Unity scene and use them independently or together.

| Module Name | Core Functionality | Key Features |
| :--- | :--- | :--- |
| **🔊 Audio Feedback System** | A centralized audio manager that dynamically assigns and plays sound effects across the entire scene. | Global Sound Library, dropdown assignment in the Inspector, instant project-wide audio swapping, looping support. |
| **💬 Interactive Tooltip System** | A UI framework that generates responsive, stylized labels when users interact with 3D objects. | Configurable hover delays, pin-to-lock mechanics, custom height offsets, auto-calibrated canvas boundaries. |

## 🚀 Getting Started

To keep things clean, all setup instructions, dependencies, and prefabs are kept inside their respective module folders. 

1. Browse to the specific module folder you want to use (e.g., `/Audio System` or `/Tooltip System`).
2. Open the specific `README.md` or `Setup.txt` inside that folder.
3. Follow the quick 5-step integration guides to drop the prefabs into your scene.
